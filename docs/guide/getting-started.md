# Getting Started

FrontendDevHelper is a browser extension that provides 35+ professional development tools right in your browser.

## Installation

### Chrome Web Store (Recommended)

1. Visit the [Chrome Web Store page](https://chrome.google.com/webstore/detail/frontenddevhelper)
2. Click "Add to Chrome"
3. The extension icon will appear in your toolbar

### Firefox Add-ons

1. Visit [Firefox Add-ons](https://addons.mozilla.org/en-US/firefox/addon/frontenddevhelper/)
2. Click "Add to Firefox"

### Manual Installation

1. Download the latest release from GitHub
2. Extract the ZIP file
3. Open Chrome and navigate to `chrome://extensions`
4. Enable "Developer mode" (toggle in top right)
5. Click "Load unpacked"
6. Select the extracted folder

## Opening the Extension

- **Click the icon** in your browser toolbar
- **Keyboard shortcut**: `Ctrl+Shift+F` (or `Cmd+Shift+F` on Mac)

## Your First Tool

Let's try the DOM Outliner:

1. Open the extension popup
2. Click "DOM Outliner" in the Inspection category
3. Or use the keyboard shortcut: `Alt+P`

You'll see colored outlines around all elements on the page, making it easy to visualize the DOM structure.

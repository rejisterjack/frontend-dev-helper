## Design System

### Colors (Catppuccin Mocha inspired)
- Base: #1e1e2e
- Surface: #313244
- Text: #cdd6f4
- Subtext: #6c7086
- Accent (Purple): #a855f7
- Accent (Blue): #6366f1
- Success: #a6e3a1
- Warning: #f9e2af
- Error: #f38ba8

### Typography
- Primary font: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto
- Monospace: 'Fira Code', 'JetBrains Mono', monospace
- Headings: 24-32px, bold
- Body: 14-16px, regular
- Captions: 12px, medium

### Icons
- Use Lucide icons or similar
- Size: 24px for UI, 48px for features
- Style: Outlined, 2px stroke

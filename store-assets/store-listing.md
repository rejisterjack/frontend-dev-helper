## Full Description

**One extension. Every visual debugging tool you actually need.**

FrontendDevHelper is the modern, unified replacement for 8–12 separate browser extensions. Built for Chrome, Brave, Edge & Firefox with Manifest V3 from day one.

### 🛠️ 19 Professional Tools Included:

**Core Inspection Tools:**
• DOM Outliner — Color-coded element outlines (Pesticide reborn)
• Spacing Visualizer — Margin & padding overlays on click
• Font Inspector — Typography analysis with source detection
• Color Picker — EyeDropper + palette extraction
• Pixel Ruler — Precise distance measurement
• Breakpoint Overlay — Viewport size & responsive breakpoints

**CSS & Layout Tools:**
• CSS Inspector — All computed properties by category
• Contrast Checker — WCAG AA/AAA accessibility compliance
• Flex/Grid Visualizer — Visualize layout structures
• Z-Index Visualizer — Stacking order with 3D view
• Live CSS Editor — Real-time CSS editing

**Analysis Tools:**
• Tech Detector — Detect frameworks & libraries (React, Vue, Tailwind, etc.)
• Accessibility Audit — WCAG validator + ARIA checker
• Site Report Generator — Comprehensive analysis & reports
• Network Analyzer — Network request monitoring
• Animation Inspector — Debug CSS animations

**Productivity Tools:**
• Screenshot Studio — Capture & annotate screenshots
• Responsive Preview — Multi-device preview
• Design System Validator — Design token consistency

### ✨ Key Features:

✅ Manifest V3 compliant — works on all modern browsers
✅ Zero data collection — everything runs locally
✅ Keyboard shortcuts — Alt+P, Alt+S, Alt+F, Alt+C, Alt+M, Alt+B
✅ Export reports — JSON, PDF, Markdown formats
✅ Dark mode — Built-in dark theme
✅ Open source — Free forever

### 🔒 Privacy First

FrontendDevHelper does NOT collect any data. All processing happens locally in your browser. No external servers, no analytics, no tracking.

### 🚀 Perfect For:

• Frontend developers debugging layouts
• UI/UX designers inspecting spacing & typography
• QA engineers checking accessibility
• Performance engineers analyzing page load
• Anyone learning CSS & HTML

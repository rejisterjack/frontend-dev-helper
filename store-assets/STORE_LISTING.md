## Short Description (132 characters max)
All-in-one dev toolkit: DOM inspector, color picker, CSS editor, Smart Suggestions, and 35+ tools for frontend developers.

## Detailed Description

```
FrontendDevHelper is the ultimate browser extension for web developers, packed with 35+ essential tools to streamline your workflow.

🔧 CORE TOOLS:
• DOM Outliner - Visualize page structure with colored outlines
• Color Picker - Pick colors from any element with palette extraction
• CSS Inspector - View computed styles by category
• Spacing Visualizer - See margins, padding, and gaps
• Font Inspector - Analyze typography with source detection
• Pixel Ruler - Precise measurements with px/rem display

⚡ PERFORMANCE:
• Network Analyzer - Monitor requests and timing
• Performance Flame Graph - Visualize JS execution
• Core Web Vitals - LCP, FID, CLS tracking
• Performance Budget Alerts - Set and monitor budgets

🎨 CSS & DESIGN:
• CSS Variable Inspector - Edit CSS custom properties live
• Flex/Grid Visualizer - Debug layouts
• Contrast Checker - WCAG AA/AAA compliance
• Z-Index Visualizer - See stacking context with 3D view
• Container Query Inspector - Debug @container rules
• Animation Inspector - Debug CSS animations

🔥 BEAST MODE (Next-Gen):
• View Transitions Debugger - Debug the View Transitions API
• Scroll-Driven Animations Debugger - Debug scroll() and view() timelines
• Smart Element Picker - Unified inspection panel
• Component Tree - React/Vue/Angular/Svelte debugging
• Storage Inspector - LocalStorage, IndexedDB, Cookies
• Screenshot Studio - Capture and annotate
• Visual Regression - Compare screenshots

🤖 SMART ANALYSIS:
• Smart Suggestions - Intelligent analysis with 50+ detection patterns
• Site Report Generator - Comprehensive audits
• Design System Validator - Check consistency

🛠️ ADVANCED:
• Accessibility Audit - WCAG validator + ARIA checker
• Form Debugger - Debug forms & validation
• Focus Debugger - Visualize focus order
• Tech Detector - Detect frameworks & libraries
• Session Recorder - Record and replay sessions

⌨️ KEYBOARD SHORTCUTS:
Alt+P: DOM Outliner | Alt+S: Spacing | Alt+C: Color Picker
Alt+F: Font Inspector | Alt+B: Breakpoints | Ctrl+Shift+P: Command Palette
Alt+Q: Container Queries

💎 FREE & PRO:
Free includes all 35+ tools with Smart analysis.
Pro unlocks team collaboration and advanced features.

Open source: github.com/rejisterjack/frontend-dev-helper
```
